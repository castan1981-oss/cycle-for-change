Strava Setup — do this once

This page trades a temporary code for the permanent refresh token the website needs. You only run it one time. Nothing here is saved or sent anywhere except Strava.

Step 1 · Create your Strava app

  1. Go to strava.com/settings/api and create an application.
  2. Set Authorization Callback Domain to localhost
  3. Copy your Client ID and Client Secret below.

Step 3 · Paste the code Strava gives back

After you authorize, Strava sends you to a localhost page that won't load — that's fine. Copy the code= value from that page's address bar and paste it here.
